🚀 PharmaShare – Medicine Sharing Platform
📌 Overview

PharmaShare is a full-stack web application designed to help users share unused medicines with people in need. It aims to reduce medicine waste and make healthcare more accessible.

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript, React
Backend: Node.js, Express.js
Database: MongoDB
Other Tools: Git, GitHub
✨ Features
🔐 User Authentication (Login / Signup)
💊 Add & Donate Medicines
🔍 Browse Available Medicines
📦 Request Medicines
📱 Responsive UI
📂 Project Structure
pharmaShare/
│── client/     # Frontend (React)
│── server/     # Backend (Node.js + Express)
│── README.md

⚙️ Installation & Setup
Clone the repository
git clone https://github.com/Ronitmaurya53/PharmaShare.git
Go to project folder
cd PharmaShare
Install dependencies
cd client
npm install

cd ../server
npm install
Run the project
# Start backend
cd server
npm start

# Start frontend
cd client
npm start
🎯 Purpose

The goal of PharmaShare is to create a community-driven platform where people can donate unused medicines instead of wasting them.
